## 二、黑盒测试（老默负责 - 前端页面流程测试）

### 10. 用户流程
| ID | 用例 | 步骤 | 预期 |
|----|------|------|------|
| U01 | 注册→激活→登录 | 注册新用户→输入卡密→登录 | 进入首页 |
| U02 | 未激活用户登录 | 登录未激活账号 | 跳转激活页 |
| U03 | 修改密码后重新登录 | 改密→退出→用新密码登录 | 成功 |
| U04 | 退出登录 | 点击退出 | 跳转登录页，token清除 |

### 11. 班级与学生流程
| ID | 用例 | 步骤 | 预期 |
|----|------|------|------|
| U05 | 创建班级 | 点击班级名→新建 | 班级列表出现新班级 |
| U06 | 切换班级 | 点击班级名→选另一个 | 学生列表切换 |
| U07 | 添加学生 | 设置页输入名字→添加 | 首页出现学生卡片 |
| U08 | 批量添加 | 设置页→批量添加→输入多行 | 多个学生出现 |
| U09 | 删除学生 | 设置页→删除 | 学生消失 |

### 12. 宠物与积分流程
| ID | 用例 | 步骤 | 预期 |
|----|------|------|------|
| U10 | 选择宠物 | 点击蛋→选宠物 | 卡片显示宠物图片 |
| U11 | 加分 | 点击学生→选加分项 | 进度条增长，food_count增加 |
| U12 | 扣分 | 撤回模式→点击学生 | food_count减少 |
| U13 | 批量加分 | 批量模式→全选→批量喂养 | 所有学生food_count增加 |
| U14 | 宠物成长 | 多次加分到阶段阈值 | 宠物图片变化 |
| U15 | 满级毕业 | 加分到100→召唤守护兽 | 获得徽章，进度重置 |

### 13. 特色功能

| U17 | 不重复点名 | 连续点名多次 | 不出现重复 |

| U20 | 徽章墙 | 点击学生徽章数字 | 弹出徽章墙 |

### 14. 小卖部流程
| ID | 用例 | 步骤 | 预期 |

## 三、边界测试（开发助理负责）

| ID | 用例 | 预期 |
|----|------|------|
| BD01 | 用户名255字符 | 正常或截断 |
| BD02 | 密码6位最短 | 通过 |
| BD03 | 密码5位 | 拒绝 |
| BD04 | 一次批量添加100个学生 | 正常 |
| BD05 | 学生名字含特殊字符 | 正常存储显示 |
| BD06 | food_count加到极大值 | 不溢出 |
| BD07 | 同时撤回已撤回的记录 | 不重复撤回 |
| BD08 | 空班级获取学生 | 返回空数组 |
| BD09 | 删除有学生的班级 | 级联删除或拒绝 |
| BD10 | 并发加分同一学生 | 数据一致 |
